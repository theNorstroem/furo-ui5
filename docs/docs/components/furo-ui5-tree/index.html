    furo-ui5-tree #

    @furo/components v1.0.0-alpha.4
    import '@furo/components/src/furo-ui5-tree.js';
    exports FuroUi5Tree js
    exports <furo-ui5-tree> custom-element-definition
    superclass LitElement
    mixes FBP

    tree navigation menu

    furo-tree renders a tree structure

    Attributes and Properties #

    tabindex #

    tabindex reflects number default: 0

    Sets the tabindex

    _searchTerm #

    default: ''



    expandDepth #

    expand-depth number default: 2

    Sets the maximal expand level relative from the current node.

    Expanding is a expensive operation.

    _searchIsActive #

    searching reflects boolean default: false

    indicator for searching. Maybe you want style your item depending on this attribute

    _tic #

    If you want to use a custom component for the tree-item, set this attribute. The default item component is furo-ui5-tree-item.

    _treeItemTepmplate #



    depth #

    depth

    Maximal depth for the tree. Default is infinite.

    qp #

    qp

    Query param to watch. If you set this attribute, the node-selected event will only be fired on ƒ-qp-in or ƒ-select-by-id. If you select an item the qp-change-request will be fired.

    rootAsHeader #

    root-as-header

    Set this flag if you do not want a header-text section.

    hideRootNode #

    hide-root-node

    Set this flag if you do not want to see the root node

    headerText #

    header-text

    Override display name from root object

    secondaryText #

    secondary-text

    Override description from root object.

    nobgonhead #

    no-bg-on-header

    disables the background color on focus, selected, … on header node

    Works only with root-as-header enabled

    focused #

    focused reflects

    indicates that the element is focused

    node-selected #

    @-node-selected → selected field

    Fired when the item gets selected, does not fire when you work with query params

    qp-change-requested #

    @-qp-change-requested → Object {&#34;this.qp&#34;: this._selectedField.id._value}

    Fired when qp mode is enabled. Nodes are only selectable with qpIn or selectById

    focusFirst #

    focusFirst() ⟹ void

    * → ƒ-focus-first

    Focuses the first node in the tree without selecting it.

    Use selectFocused to select the focused node.

    The tree item will also scrollIntoViewIfNeeded() (on all platforms which support this method).



    focusLast #

    focusLast() ⟹ void

    * → ƒ-focus-last

    Focuses the last node in the tree without selecting it.

    Use selectFocused to select the focused node.

    The tree item will also scrollIntoViewIfNeeded() (on all platforms which support this method).



    focusNext #

    focusNext() ⟹ void

    * → ƒ-focus-next
