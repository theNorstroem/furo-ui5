    Methods #

    triggerNavigation #

    triggerNavigation(key `` ) ⟹ void

    `` → ƒ-trigger-navigation

    Connect your navigation pad.

    Default mappings are:

    • Enter => selectFocused
    • ArroLeft => focusPrevious
    • ArroRight => focusNext
    • Home => focusFirst
    • End => focusLast
    • Escape => closeFocused

    To disable a function, just add the ignored-keys to furo-navigation-tabs.

    • key

    closeFocused #

    closeFocused() ⟹ void

    * → ƒ-close-focused

    Closes the focused tab.



    focusNext #

    focusNext() ⟹ void

    * → ƒ-focus-next

    Focuses the next tab. If you are on the last tab, the frist tab will be selected

    focusFirst() ⟹ void

    focusLast() ⟹ void

    * → ƒ-focus-last

    focusPrevious #

    * → ƒ-focus-previous
