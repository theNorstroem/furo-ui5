    @furo/ui5 #

    @furo/ui5 v1.0.0-rc.2 SAP UI5 Web Components data bindings for furo

    Installation #

    npm:

    1

    npm i -S @furo/ui5`

    cdn:

    1
2
3

    <script type="module"
        src="https://cdn.jsdelivr.net/npm/@furo/precompiled@1.0.0-rc.2/dist/typerenderer.js">
</script>
